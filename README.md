 User dashboard client for blockops

## Language
- React
- Typescript

## Requirements

### Application

-   React (Version >= 18.0)

-   NPM  (Version >= 8.3.0)

-   Tailwindcss (Version >= 3.0)

-   Jest (Version >= 29.0)

-   TypeScript (`npm install -g typescript` || `yarn global add typescript`)
